💛 温馨提示：移动端可将整个编辑器固定在顶部，高度设置为屏幕的一半，以防止不同机型、不同浏览器在软键盘弹起时不一致的表现。 
  new Vditor('vditor', {
    toolbarConfig: {
      pin: true
    },
    counter: {
      enable: true
    },
    height: window.innerHeight / 2,
    toolbar: [
      'emoji',
      'link',
      'upload',
      'edit-mode',
      {
        name: 'more',
        toolbar: [
          'insert-after',
          'fullscreen',
          'preview',
          'info',
          'help',
        ],
      },
    ],
  })
