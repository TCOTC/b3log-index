Symphony

[ˈsɪmfəni]，n.交响乐

下一代的社区系统，为未来而构建

  • 实现了面向内容讨论的论坛
  • 实现了面向知识问答的社区
  • 包含了面向用户分享、交友、游戏的社交网络
  • 社区版 100% 开源
  • 使用 Java 的企业搭建社区的不二之选！
下载背景、功能及技术架构简介.ppt

人性化实时交互

  • 回帖提交后其他浏览者可以不刷新页面就实时看到你的回帖
  • 回复/引用可以在当前位置进行展开浏览，也可以跳转到回帖处，并支持跨分页跳转
  • 帖子、回帖和评论内容中如果包含其他帖子、回帖和评论的 URL，将自动建立双向链接
  • 实时消息通知，细致分类：回帖、回复、提及、关注、积分、同城、系统
  • 支持离线桌面提醒，关闭浏览器页签也可以收到消息通知

搜索引擎优化

  • 标签、领域可以自定义链接，并且可以单独配置 meta title/keywords/description
  • 后端 Markdown 渲染，输出爬虫可读的 HTML
  • 精细化的 Title、H1 生成，确保全站唯一性
  • 充分使用现代化 HTML 语义标签和结构化数据 JSON-LD
  • 通过配置集成 Elasticsearch

主持人（版主）

  • 主持人类似传统论坛的版主
  • 可管理标签/领域关联的帖子、回帖和评论
  • 可配置是否能由主持人再授权给其他用户成为主持人

大后台

  • 帖子管理、回帖管理、评论管理
  • 聊天室管理、清风明月管理
  • 文件管理、审核管理
  • 用户管理、角色管理、主持人管理
  • 领域管理、标签管理
  • 违禁词管理、邀请码管理
  • 广告管理、举报管理
  • 审计日志、数据分析

全面的用户行为量化

  • 发布帖子、回帖和评论统计
  • 帖子、主页浏览量统计
  • 帖子、用户收藏和关注统计
  • 打赏、感谢、点赞统计
  • 提问、回答、采纳统计
  • GitHub 仓库、星数、粉数统计
  • 举报、被举报、点踩统计
  • 使用时长、贡献点统计

提供多款主题

  • Classic: Light | Dark
  • Mobile - 移动端专属主题: Light | Dark
  • 知乎: Light | Dark
  • V2EX: Light | Dark
  • 可前往链滴 - 设置 - 界面体验
